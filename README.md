# Premium Portfolio - Lokesh Saini

A modern, high-performance personal portfolio website designed with a premium aesthetic, featuring glassmorphism, responsive layouts, and smooth animations.

## 🚀 Live Demo
Check out the live site: [https://golden-pithivier-d63c86.netlify.app/](https://golden-pithivier-d63c86.netlify.app/)

## ✨ Key Features
- **Premium UI/UX**: Modern glassmorphism design with a custom Indigo & Slate color palette.
- **Responsive Design**: Fully optimized for mobile, tablet, and desktop screens.
- **Dynamic Interactions**:
  - **Scroll Reveal**: Sections slide into view as you scroll.
  - **Hero Animations**: Sequenced fade-in effects for the introductory section.
  - **Interactive Project Cards**: 3D-lift effects and tech-stack pills.
- **Dark/Light Mode**: Seamless theme switching with persistent storage.
- **Functional Contact Form**: Integrated with Formspree for direct email notifications.

## 🛠️ Tech Stack
- **Structure**: Semantic HTML5
- **Styling**: Vanilla CSS3 (Custom Variables, Flexbox, CSS Grid)
- **Logic**: Vanilla JavaScript (ES6+, Intersection Observer API)
- **Fonts**: Google Fonts (Outfit & Inter)

## 📁 Project Structure
```text
portfolio-project/

```

## 💻 Getting Started
1. Clone the repository:
   ```bash
   git clone https://github.com/2024ucp1505/portfolio-project.git
   ```
2. Navigate to the project directory:
   ```bash
   cd portfolio-project
   ```
3. Open `index.html` in your browser (preferably through a local server like Live Server).
